A sleek and interactive music player web app built using HTML, CSS, and JavaScript.
Designed with smooth animations, responsive UI elements, and dynamic per-track backgrounds to enhance the visual and audio experience.

Features:
Dynamic UI:

Each track has its own album art and background image.

Animated, floating background effect for a modern, aesthetic feel.

Clean and responsive layout with hover animations.

Music Controls

Play / Pause

Next Track

Previous Track

Auto-play next track when the current one ends

Sliders:

Seek slider to jump to any part of the track

Volume slider for adjusting sound level

Live updates for:

Current time

Total duration

Audio Features:

Uses the built-in <audio> element

Smooth track loading

Displays “Now Playing” status with track number
